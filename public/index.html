API Basejump: URL shortener

Example usage:


https://urlstner.herokuapp.com/new/https://www.youtube.com/watch?v=Jl0G8KqpBV0

https://urlstner.herokuapp.com/new/https://www.youtube.com/watch?v=rM3PTRtOyVM

Example output:


{
"original_url": "https://www.youtube.com/watch?v=Jl0G8KqpBV0",
"short_url": "https://urlstner.herokuapp.com/323756"
}
